# LibraryProgram

#주제: 도서관 관리 프로그램

##요구사항
---
1. 책 이름, 저자, 출판사를 입력받는다. (5개)
2. 등록된 책 정보 출력한다.
3. 대출을 할 때 자신의 이름을 입력하면 책 정보가 나옴.
4. 책이름을 입력을 하면 대출자, 책이름, 저자, 출판사 출력됨
5. 대출할거냐는 MSG가 나왔을 때 y를 입력하면 대출, n를 입력하면 취소되었습니다.
6. 그리고 대출한 책이 있으면 이미 대출되어 있는 책이라 MSG출력
7. 반납을 할 때
